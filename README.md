# omdb-app-rails

Aplicacion de busqueda de peliculas utilizando como fuente la api: http://www.omdbapi.com, permite la busqueda por nombre y por año; 
ofrece un panel de metricas.

## Comenzando 🚀

- Desarrollador en Rails 6.0.0
- Ruby 3.0.0
